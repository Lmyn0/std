# personal study repository
# Claude API Prompt Engineering Study 

이 레포지토리는 **Claude API**를 활용한 프롬프트 엔지니어링 실습과 AI 모델 연동 과정을 기록하는 공간입니다. AI 직무 역량을 키우기 위한 기술적 기반을 다지는 프로젝트입니다.

## 주요 실습 내용
- **API 보안 관리**: `.env` 및 `.gitignore`를 활용하여 API 키를 안전하게 관리하는 법 습관화.
- **System Prompt 최적화**: 모델의 페르소나와 출력 규칙을 설정하여 원하는 답변을 유도하는 기법 실습.
- **Python 연동**: `anthropic` 라이브러리와 `python-dotenv`를 활용한 효율적인 API 호출 함수 구현.

## 프로젝트 구조
- `completion.ipynb`: Claude API 호출 기초 및 프롬프트 엔지니어링 실습 노트북
- `.env`: API 키 저장 파일 (보안상 업로드 제외)
- `.gitignore`: GitHub 업로드 제외 목록 설정 파일
